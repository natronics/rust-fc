A sequence error message.

When we miss a packet or get an out of order packet we should log that for future analysis. This stores a error and builds a log-able message.

Example

use rust_fc::io;

let seqerror = io::SequenceError {
    port: 35020,
    expected: 12345,
    received: 12349,
};

// Now you can log or send the message somewhere
